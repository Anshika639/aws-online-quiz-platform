# AWS Online Quiz Platform

## Project Overview

AWS Online Quiz Platform is a cloud-based web application developed using HTML, CSS, JavaScript, and AWS services. The platform allows users to answer multiple-choice questions, receive instant score feedback, and interact with a user-friendly quiz interface.

## Features

* Multiple-choice quiz questions
* Instant score calculation
* Interactive and responsive user interface
* Cloud-based deployment using AWS
* Easy navigation and user-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript
* AWS
